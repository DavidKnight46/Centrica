Centrica excerise

Pre-requites:

The following software must be installed to run the standalone Java Application:
Maven
Java1.8

Installing:

Unzip the file onto C:\ as default or any location desired.

Deployment:

Run Maven using command line interface "mvn pakage" to compile and build applications and run unit tests.

Run:

From command line type "java -jar excerise-0.0.1-SNAPSHOT-jar-with-dependencies.jar" to run the application

Assumptions:

None.
